## 2. 프로젝트 실행 방법

### 로컬에서 실행
> 이 프로젝트는 로컬서버에서 실행됩니다. 아래의 단계를 따라주세요. 

#### 1. 저장소 복제
해당 저장소를 로컬로 복제합니다.

```
git clone [깃허브 저장소 주소]
```

#### 2. 패키지 설치
필요한 패키지를 설치합니다.

```
npm install
```

#### 3. 로컬 서버 실행
명령어를 입력해 서버를 실행합니다.

```
npm start
```
서버를 실행했다면,  http://localhost:3000 주소로 접속하여 앱을 볼 수 있습니다.
